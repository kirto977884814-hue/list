# 快速测试指南

## 测试准备

### 1. 创建测试Excel文件

**文件1：members.xlsx（会员名单）**

创建一个Excel文件，包含以下内容：

| 用户ID | 用户昵称 |
|--------|----------|
| 1001   | 张三     |
| 1002   | 李四     |
| 1003   | 王五     |
| 1004   | 赵六     |
| 1005   | 钱七     |

**文件2：view_test.xlsx（观看名单）**

创建另一个Excel文件，包含以下内容：

| 用户ID |
|--------|
| 1001   |
| 1003   |
| 1005   |

### 2. 启动系统

**Windows**：
```bash
双击 start.bat
```

**Mac/Linux**：
```bash
bash start.sh
```

或手动启动：
```bash
# 终端1 - 启动后端
cd backend
npm start

# 终端2 - 启动前端
cd frontend
npm run dev
```

### 3. 访问应用

浏览器打开：http://localhost:5173

## 测试步骤

### 测试1：导入会员名单

1. 在"会员名单导入"卡片中
2. 拖拽或点击上传 `members.xlsx`
3. 检查是否显示："会员名单导入成功，共导入 5 名会员"
4. 查看统计卡片是否显示"当前会员总数: 5人"

**预期结果**：✅ 导入成功，显示5名会员

### 测试2：导入观看名单

1. 在"观看记录名称"输入框中输入："测试直播观看记录"
2. 拖拽或点击上传 `view_test.xlsx`
3. 检查是否显示成功提示，包含：
   - 总会员: 5
   - 已观看: 3
   - 未观看: 2

**预期结果**：✅ 导入成功，统计正确

### 测试3：查看结果

系统应自动展示：

**统计卡片**：
- 总会员数: 5

- 未观看: 2

**警告信息**：
"共有 2 名会员未观看直播"

**结果表格**：
| 用户ID | 用户昵称 |
|--------|----------|
| 1002   | 李四     |
| 1004   | 赵六     |

**预期结果**：✅ 正确显示未观看的会员（李四、赵六）

## 常见问题排查

### 问题1：后端启动失败
**检查**：
- Node.js是否安装：`node --version`
- 依赖是否安装：`cd backend && npm install`
- 端口3001是否被占用

### 问题2：前端启动失败
**检查**：
- 依赖是否安装：`cd frontend && npm install`
- 端口5173是否被占用

### 问题3：导入失败
**检查**：
- Excel文件格式是否正确（.xlsx或.xls）
- 列名是否严格匹配
- 文件是否损坏

### 问题4：无法连接后端
**检查**：
- 后端服务是否运行
- 浏览器控制台是否有CORS错误
- API地址配置是否正确（frontend/.env）
