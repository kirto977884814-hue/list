# 使用说明

## 首次使用

### 1. 准备Excel文件

**会员名单文件**（必须包含以下两列，列名严格一致）：
```
用户ID    用户昵称
1001      张三
1002      李四
1003      王五
```

**观看直播名单文件**（必须包含"用户ID"列，列名严格一致）：
```
用户ID
1001
1003
```

### 2. 启动系统

**Windows系统**：
双击运行 `start.bat`

**Mac/Linux系统**：
```bash
bash start.sh
```

或手动启动：
1. 终端1：`cd backend && npm start`
2. 终端2：`cd frontend && npm run dev`

### 3. 访问应用

打开浏览器访问：http://localhost:5173

## 操作流程

### 步骤1：导入会员名单

1. 在"会员名单导入"卡片中，点击上传区域或拖拽文件
2. 选择会员名单Excel文件
3. 系统会显示导入成功的会员总数

**注意**：
- 会员名单只需导入一次，系统会保存
- 重新导入会覆盖之前的会员数据

### 步骤2：导入观看直播名单

1. 输入观看记录名称（如："2024年1月15日直播"）
2. 在"观看名单导入"卡片中上传观看名单Excel文件
3. 系统自动进行对比分析

### 步骤3：查看结果

系统会自动展示：
- 总会员数
- 已观看人数
- 未观看人数
- 未观看会员的详细列表（用户ID + 用户昵称）

## 常见问题

### Q1: 上传Excel后提示"导入失败"
**检查项**：
- Excel文件格式是否正确（.xlsx或.xls）
- 列名是否严格为"用户ID"和"用户昵称"（会员名单）
- 列名是否严格为"用户ID"（观看名单）
- 数据是否为空

### Q2: 导入后看不到结果
**解决方法**：
- 确保先导入会员名单，再导入观看名单
- 刷新页面重新尝试

### Q3: 如何重新导入会员名单？
直接重新上传会员名单文件即可，会自动覆盖旧数据

### Q4: 数据会丢失吗？
- 会员名单：保存在数据库中，重启不会丢失
- 观看记录：会保存历史记录
- 数据库文件：`backend/database.db`

### Q5: 如何清空所有数据？
删除 `backend/database.db` 文件，重启后端服务器即可

## 技术支持

如遇问题，请检查：
1. 后端服务是否正常运行（http://localhost:3001）
2. 前端服务是否正常运行（http://localhost:5173）
3. 浏览器控制台是否有错误信息
4. 网络连接是否正常
