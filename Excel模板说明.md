# Excel文件模板说明

## 模板1：会员名单（members.xlsx）

### 必需列：
- **用户ID**（文本或数字）
- **用户昵称**（文本）

### 示例数据：

| 用户ID | 用户昵称 |
|--------|----------|
| 1001   | 张三     |
| 1002   | 李四     |
| 1003   | 王五     |
| 1004   | 赵六     |
| 1005   | 钱七     |

### 注意事项：
1. 第一行必须是表头，列名必须严格为"用户ID"和"用户昵称"
2. 用户ID不能重复
3. 用户昵称不能为空
4. 支持xlsx和xls格式

---

## 模板2：观看直播名单（view_records.xlsx）

### 必需列：
- **用户ID**（文本或数字）

### 示例数据：

| 用户ID |
|--------|
| 1001   |
| 1003   |
| 1005   |

### 注意事项：
1. 第一行必须是表头，列名必须严格为"用户ID"
2. 只需要包含观看了直播的用户ID
3. 用户ID必须在会员名单中存在
4. 支持xlsx和xls格式

---

## 如何创建Excel模板

### 方法1：使用Microsoft Excel
1. 新建Excel文件
2. 在第一行输入表头（用户ID、用户昵称 或 用户ID）
3. 从第二行开始输入数据
4. 保存为 .xlsx 或 .xls 格式

### 方法2：使用WPS表格
1. 新建表格
2. 在第一行输入表头
3. 输入数据
4. 保存为 Excel 格式

### 方法3：使用Google Sheets
1. 新建表格
2. 输入表头和数据
3. 下载为 Excel (.xlsx) 格式

---

## 完整示例

假设有以下会员：
- 1001 张三
- 1002 李四
- 1003 王五
- 1004 赵六
- 1005 钱七

某次直播观看的人员：
- 1001 张三
- 1003 王五

### 会员名单文件（members.xlsx）：
| 用户ID | 用户昵称 |
|--------|----------|
| 1001   | 张三     |
| 1002   | 李四     |
| 1003   | 王五     |
| 1004   | 赵六     |
| 1005   | 钱七     |

### 观看名单文件（view_20240115.xlsx）：
| 用户ID |
|--------|
| 1001   |
| 1003   |

### 对比结果：
未观看的会员：
- 1002 李四
- 1004 赵六
- 1005 钱七

## 常见错误

### 错误1：列名不匹配
❌ 错误：
| ID    | 名称    |
|-------|---------|
| 1001  | 张三    |

✅ 正确：
| 用户ID | 用户昵称 |
|--------|----------|
| 1001   | 张三     |

### 错误2：缺少必需列
❌ 错误（缺少用户昵称）：
| 用户ID |
|--------|
| 1001   |

✅ 正确：
| 用户ID | 用户昵称 |
|--------|----------|
| 1001   | 张三     |

### 错误3：数据为空
❌ 错误：
| 用户ID | 用户昵称 |
|--------|----------|

✅ 正确：
| 用户ID | 用户昵称 |
|--------|----------|
| 1001   | 张三     |
