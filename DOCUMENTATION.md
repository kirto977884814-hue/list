# 直播观看对比系统

一个简洁的纯前端Web应用，用于对比会员名单和观看直播名单，自动找出未观看的会员。

## 在线使用

**访问地址**：https://kirto977884814-hue.github.io/list/

直接打开链接即可使用，无需安装任何东西！

## 功能特点

- ✅ 纯前端应用，数据存储在浏览器本地
- ✅ 导入会员名单（Excel：用户ID + 用户昵称）
- ✅ 导入观看直播名单（Excel：用户ID）
- ✅ 自动对比并展示未观看的会员
- ✅ 简洁易用的界面

## 使用方法

### 第一步：导入会员名单（首次使用）

1. 准备Excel文件，包含两列：
   - **用户ID**
   - **用户昵称**

2. 点击"导入会员名单"按钮上传

3. 导入成功后会自动折叠，需要修改时点击展开即可

### 第二步：导入观看直播名单

1. 准备Excel文件，包含一列：
   - **用户ID**

2. 点击"点击上传观看名单"按钮

3. 系统自动对比并显示结果

### 第三步：查看结果

- 总会员数
- 未观看人数
- 未观看会员详细名单

## Excel文件格式

### 会员名单（members.xlsx）

| 用户ID | 用户昵称 |
|--------|----------|
| 1001   | 张三     |
| 1002   | 李四     |
| 1003   | 王五     |

### 观看直播名单（view.xlsx）

| 用户ID |
|--------|
| 1001   |
| 1003   |

### 对比结果

显示未观看的会员：
- 1002 李四

## 技术栈

- **前端框架**：React + Vite
- **UI组件**：Ant Design
- **Excel处理**：xlsx（浏览器端）
- **数据存储**：localStorage（浏览器本地存储）

## 注意事项

- 数据保存在浏览器本地，清除浏览器数据会丢失
- 建议使用同一个浏览器访问
- 会员名单只需导入一次，会保存在浏览器中
- 每次直播后只需上传观看名单即可

## 本地开发

如果想本地运行：

```bash
# 安装依赖
cd frontend
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
